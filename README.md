# DocuSecure

A simple yet powerful document signing platform built with Symfony and React.

## What it does

DocuSecure helps businesses get documents signed online. No more printing, signing, scanning, and emailing. Upload your document, add who needs to sign it, and the system handles the rest.

## Main features

- **Safe document storage**: Keep all your documents secure in one place
- **Flexible signing process**: Choose who signs and in what order
- **Digital signatures**: Legally valid signatures that can't be faked
- **Templates**: Save common documents to reuse later
- **Complete history**: See who did what and when
- **Smart permissions**: Control who can see and do what
- **Automatic emails**: Everyone gets notified when action is needed
- **API**: Connect with your other tools and systems

## Tech stack

- **Backend**: Symfony 6, PHP 8.1
- **API**: JWT for secure access
- **Frontend**: React
- **Database**: PostgreSQL
- **Speed improvements**: Redis
- **File storage**: Works with S3 and similar services
